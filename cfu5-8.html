    Piggy Bank Calculator

    About CFU 5

    This interactive Python program helps users, especially kids, count the total value of their coins stored in a piggy bank. The script starts with a friendly greeting and prompts the user to input the number of pennies, nickels, dimes, quarters, loonies, and toonies they have. It calculates the total amount in cents, then converts it into dollars and cents for a clear display. Additionally, the program provides a breakdown of the remaining cents into quarters, dimes, nickels, and pennies. This project is a fun way to practice working with user input, basic arithmetic, and conditional statements in Python.

    Grubhub Splitter

    About CFU 8

    This Python program simulates a scenario where users split the bill for a food delivery order. It begins by asking if the user ordered delivery, providing a friendly response based on the answer. If the user confirms a delivery, the program proceeds to gather details such as the total cost of the order and the number of people sharing the bill. This project demonstrates the use of user input, conditionals, error handling, and simple arithmetic calculations in Python.
