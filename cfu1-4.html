    Python Introduction Program

    About CFU 1

    In this Python program, I start by asking for your first and last names, then combine them to form your full name. Once I have your full name, I greet you personally, making it feel like a friendly conversation. After a brief pause, I ask you about your favorite color and respond with a compliment, letting you know I like it too. Next, I ask about your favorite school subject, giving a small pause for a natural flow. When you share your favorite subject, I enthusiastically say that I like it as well, making it seem like we have a lot in common. I used pauses (time.sleep()) to make the conversation feel smoother and more engaging. This program shows how I can use basic Python techniques to make our interaction fun and friendly!

    CFU 2


    Age Calculator & Student
    Journey Feedback Program

    About CFU 2

    In this program, I created a simple interactive script that asks for the user’s first name, last name, and age. Depending on their age, the program responds differently. If the user is younger than 21, it tells them how many years they have left until they turn 21. If they’re 21 or older, it calculates how many years it’s been since they turned 21. I also ask the user how many years they’ve been a student and include a fun message reflecting on their time in school. This little project helps practice user input, conditional statements, and string formatting!

    Friendly Python Calculator

    About CFU 3

    In this playful Python program, I created a user-friendly calculator that performs basic arithmetic operations like addition, subtraction, multiplication, division, and modulus. It greets the user warmly, asks for two pairs of numbers, and then displays the results of each calculation with clear and informative output. The program even handles division by zero gracefully, providing a helpful message instead of crashing. After performing the calculations, it wraps up with a friendly farewell message, making the whole experience more engaging!

  • CFU 4


    Quadratic Equation Solver

    About CFU 4

    This program is an interactive Python script that helps users solve quadratic equations. It starts by greeting the user and asking for their name, followed by prompts to enter the coefficients of the equation. The script then checks whether the equation has real roots by evaluating the discriminant. If real roots exist, it calculates and displays them. Otherwise, it informs the user that there are no real roots. This project is a great way to practice working with user input, conditionals, and mathematical operations in Python.
